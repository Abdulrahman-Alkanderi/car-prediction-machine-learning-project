# car-prediction-machine-learning-project
Built an end to end machine learning regression project to predict used car prices using a Kaggle dataset.


#overview
- a machine learning project that predicts used cars price using a kaggle dataset, I used a linear regressin, Support Vector Regression, and Random Forest model.
  
#outcome

| Model             | MAE      | RMSE     |
| ----------------- | -------- | -------- |
| Linear Regression | **3.02** | **3.97** |
| SVR (tuned)       | 3.37     | 5.16     |
| Random Forest     | **2.95** | 4.46     |

#parameters

- car age
- kilometers driven
- fuel type
- transmission
- ownership history
- present showroom price
